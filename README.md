# Vanica Events - Modern Event Management System

A complete redesign of the Vanica Doctor Appointment Management System, now transformed into a modern, responsive event management platform using React, TypeScript, and cutting-edge design patterns.

## 🚀 Features

### Frontend (React + TypeScript)
- **Modern Homepage**: Beautiful hero section with animations inspired by premium design patterns
- **Event Booking System**: Multi-step booking form with real-time validation
- **Admin Dashboard**: Comprehensive dashboard for managing events and bookings
- **Authentication**: Secure login system with modern UI
- **Responsive Design**: Fully responsive across all devices
- **Smooth Animations**: Using Motion/React for delightful user interactions

### Design System
- **UI Components**: Built with Radix UI primitives and Tailwind CSS
- **Color Scheme**: Pink and rose gradient themes with professional aesthetics
- **Typography**: Modern font hierarchy with gradient text effects
- **Interactive Elements**: Hover effects, micro-interactions, and smooth transitions

## 🛠 Tech Stack

### Frontend
- **React 18** with TypeScript
- **Vite** for build tooling
- **Tailwind CSS** for styling
- **Motion/React** for animations
- **Radix UI** for accessible components
- **React Router DOM** for navigation
- **React Hook Form** for form handling

### UI Components
- Button, Card, Input, Label, Textarea
- Modern form controls with validation
- Responsive grid layouts
- Animated backgrounds and particles

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Navigate to the project directory:
```bash
cd Vanica
```

2. Install dependencies:
```bash
npm install --legacy-peer-deps
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit:
```
http://localhost:3000
```
